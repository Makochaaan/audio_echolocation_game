Echolocation Maze (エコーロケーション迷路 プロトタイプ)

視覚ではなく「音」を頼りに迷路を探索する、ターン制のステルス・探索ゲームのプロトタイプです。 プレイヤーは白杖の代わりに自身の足音を使い、壁からの反響音（エコーロケーション）を聞き分けて進むべき道を判断します。また、暗闇の中を気まぐれに逃げ回る「猫（ゴール）」を、鳴き声を頼りに追い詰める立体音響を活かしたゲームプレイが特徴です。

※ ステレオイヤホン・ヘッドホンの着用 を強く推奨します。

特徴

ターン制・グリッド移動: 1歩進む、または方向転換するごとに1ターンを消費します。

自動反響システム (ソナー): 1歩進んで足音が鳴ると同時に、周囲8方向へ見えない光線を飛ばし、壁との距離に応じた遅延と完全な3Dサウンドで反響音を返します。

動くゴール: プレイヤーが1ターン行動するごとに、ターゲットである「猫」もランダムに1マス移動し、鳴き声を発します。

触覚（衝突）フィードバック: 障害物にぶつかると音が鳴り、特定のコンテナにぶつかると数ターン持続するノイズが発生して目印（あるいは邪魔なノイズ）となります。

💻 Unityでの起動の仕方

1. 必要な環境

Unity 2022.3 LTS 以上（推奨）

ステレオ出力が可能なオーディオデバイス（イヤホン等）

2. プロジェクトの導入と起動

このリポジトリをローカルにクローン、またはZIPでダウンロードして展開します。

git clone [https://github.com/YourUsername/YourRepositoryName.git](https://github.com/YourUsername/YourRepositoryName.git)


Unity Hub を開き、「追加 (Add)」からダウンロードしたプロジェクトのフォルダを選択します。

リストに追加されたプロジェクトをクリックしてUnityエディタを起動します。

3. Build Settings の確認

シーン遷移（スタート画面からゲーム画面への移行など）を正常に行うため、シーンの登録が必要です。

Unityエディタ上部のメニューから File > Build Settings... を開きます。

Scenes In Build のリストに以下の順序でシーンが登録されているか確認します（なければ Assets/Scenes フォルダからドラッグ＆ドロップしてください）。

0: StartScene

1: GameScene

Build Settings ウィンドウを閉じます。

4. ゲームのプレイ

プロジェクトウィンドウ（Project）の Assets/Scenes フォルダにある StartScene をダブルクリックして開きます。

エディタ上部の ▶（再生）ボタン を押します。

Press Enter to Start と表示されたら、Enter キーを押してゲームを開始してください。

⚙️ 内部で設定する項目（インスペクター設定一覧）

ゲーム内の挙動や難易度は、各オブジェクトにアタッチされたスクリプトのパラメータ（Inspectorウィンドウ）から簡単に調整可能です。

🏷️ 必須のTag（タグ）設定

このゲームでは、オブジェクトの判定に以下のカスタムタグを使用しています。Edit > Project Settings > Tags and Layers から設定されている必要があります。

Ground : 通常の床（土の音）

Metal : 金属の床（鉄板の音）

Container : ぶつかると持続ノイズを発する特殊な障害物

📦 スクリプト別 パラメータ一覧

1. PlayerController (アタッチ先: Player)

プレイヤーの移動、足音、衝突時の処理、およびターン進行を管理します。

Move Time (デフォルト値: 0.2) 1マス移動、または方向転換にかかる時間（秒）。小さいほどキビキビ動きます。

Grid Size (デフォルト値: 1.0) 1マスのサイズ。通常はモデルのサイズ（1.0）に合わせます。

Ground Sound Ground タグの床を歩いた時の足音 (AudioClip)

Metal Sound Metal タグの床を歩いた時の足音 (AudioClip)

Bump Sound 壁や障害物にぶつかった瞬間の衝撃音 (AudioClip)

Sustain Noise Sound Container タグの障害物にぶつかった後に鳴り続けるノイズ音 (AudioClip)

Sustain Duration Turns (デフォルト値: 2) コンテナ衝突後のノイズが鳴り続けるターン数。0 にすると持続しません。

Consume Turn On Bump (デフォルト値: True) 壁にぶつかって移動がキャンセルされた際も、1ターン消費して猫を動かすか。

Cat Goal 連携する猫（ゴール）オブジェクトのスクリプトへの参照。必ずアタッチしてください。

2. Echolocation (アタッチ先: Player)

反響定位（ソナー）のシステムです。足音が鳴った際、またはSpaceキーを押した際に発動します。

Echo Sound 壁から跳ね返ってくる反響音（水滴の音や短い電子音を推奨）

Max Distance (デフォルト値: 20) 音が届く（壁を検知できる）最大距離。

Sound Speed (デフォルト値: 10) ゲーム内の音速。数値を小さくすると、遠くの壁からの反響が遅く（時間差が大きく）返ってきます。

Volume Rolloff Curve (デフォルト値: Linear) 反響音が距離に応じてどれくらい小さくなるかを決める減衰カーブ。

3. CatGoalController (アタッチ先: CatGoal)

プレイヤーの行動に連動して動き回るゴールの処理です。

Move Time (デフォルト値: 0.3) 1マス移動にかかる時間（アニメーション速度）。

Grid Size (デフォルト値: 1.0) 1マスのサイズ。プレイヤーの Grid Size と揃えます。

Meow Sound 猫が動いた時（1ターンごと）に発する鳴き声 (AudioClip)

(AudioSource設定) Spatial Blend を 1.0 にし、完全な3Dサウンドとして設定しておく必要があります。

4. CameraController (アタッチ先: Main Camera)

プレイヤーを一定の距離から追従します。

Target (デフォルト値: Player) 追従する対象のTransform。

Offset (デフォルト値: X:0, Y:6, Z:-6) ターゲットからどれだけ離れてカメラを配置するかのXYZ座標。

5. InactivityTimer (アタッチ先: GameScene内の任意のオブジェクト)

無操作状態が続いた際に、スタート画面へ自動で戻るタイムアウト機能です。

Timeout Seconds (デフォルト値: 60) 操作が行われないままこの秒数が経過すると、スタートシーンに遷移します。

🎮 操作方法 (GameScene)

W / S / A / D または 矢印キー:

プレイヤーが向いている方向と同じキーを押す: 1マス前進する（1ターン消費）

プレイヤーが向いている方向と違うキーを押す: その方向へ振り向く（1ターン消費）

Space キー:

手動でソナー（反響音テスト）を発動します（ターンは消費しません）。
